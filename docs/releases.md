# Desktop releases

Aanpak voor macOS en later Windows:

- Beta tags: `desktop-v0.2.0-beta.1`, `desktop-v0.2.0-beta.2`
- Stabiele tags: `desktop-v0.2.0`
- GitHub Releases bewaren release notes en build-artifacts.
- Installers zoals `.dmg`, `.msi` en `.exe` worden niet gecommit.

Zo kunnen we altijd terug naar een exacte versie zonder build-output in de repo te bewaren.
